# FirebaseClass2CSClub


Make sure to do the following (otherwise nothing will work!):

1. Go through the whole project creation (doing **npm create vite**)
2. Note, you should see some instructions that follow the lines of 
  cd CSClub_Exampler                        
  npm install
  npm run dev 
  **Don't run it yet, but keep these instructions in mind**
3. Paste the following dependencies (separately): **npm install firebase**, **npm install react-bootstrap bootstrap**, **npm install bootstrap**
4. If you haven't done so already, make a firebase account.
5. Go to Project Overview, then to Project Settings.

You will see some code (you should be on a screen like this:  ![Screenshot 2024-11-24 at 7 28 17 PM](https://github.com/user-attachments/assets/55d826dd-9c08-4136-a010-24798d0fb860) )

Just copy paste the code that you see into cs_club_firebase/src/firebase/firebase.jsx, on line 12 specifically.

Run the instructions you got from step 2, and things should work.
